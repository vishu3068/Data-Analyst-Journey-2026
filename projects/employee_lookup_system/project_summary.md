**Employee Lookup System using VLOOKUP**

# Project Overview:

This project was created using Microsoft Excel to quickly find employee details based on Employee ID.

# The system uses:

VLOOKUP Function
Data Validation Dropdown
Employee Database
Salary Lookup
Working Hours Lookup

This project simulates a real-world HR system where employee information can be retrieved instantly.

# Objective

To learn how organizations use Excel to search and retrieve employee information from large datasets.

# Features
# Employee ID Dropdown

Users can select an Employee ID from a dropdown list.

# Salary Lookup

The system automatically displays the employee's salary.

# Working Hours Lookup

The system automatically displays the employee's weekly working hours.

# Dynamic Results

Changing the Employee ID instantly updates the report.

# Formula Used
 ##  Salary Lookup

 ## Working Hours Lookup
=VLOOKUP(I8,A2:D21,4,FALSE)
