# Student Result Analysis

## Objective

Analyze student marks and identify performance statistics.

## Dataset

20 students
Marks between 0-100

## Analysis Performed

- Pass Count
- Fail Count
- Highest Marks
- Lowest Marks
- Average Marks
- Pass Percentage

## Key Insights

- Total Students: 20
- Pass Count: 16
- Fail Count: 4
- Pass Percentage: 80%

- Microsoft Excel
